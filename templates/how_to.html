How to Use

The Power of QR Codes

At the heart of FileBhejo is QR code technology, making file sharing quick and secure.

User Types

FileBhejo caters to two types of users: Admin Users and End Users.

Admin User - New

If you're a new admin user, here's how to get started:

  1. Create an admin account on www.filebhejo.in.
  2. Log in to your account.
  3. Download the generated QR code and print it.
  4. Affix the QR code in a suitable location.
  5. You're now ready to receive files via the QR code.
  6. Await incoming files.
  7. Received files will appear in your admin panel.
  8. Print or download them as needed.
  9. You also have the option to delete files.

Admin User - Existing

For existing admin users, the process is simpler:

  1. Visit www.filebhejo.in.
  2. Log in to your existing admin account.
  3. Wait for incoming files.
  4. Received files will be displayed in your admin panel.
  5. Print, download, or delete files as required.

End User

Now, let's see how end users can benefit from FileBhejo:

  1. Visit one of our designated FileBhejo locations.
  2. Open a QR code scanner app or use Google Lens.
  3. Scan the FileBhejo QR Code displayed at the location.
  4. Follow the link provided.
  5. Click the 'Upload' button and select the file you wish to send.
  6. Confirm the upload.
  7. You're done! Your file is on its way.
